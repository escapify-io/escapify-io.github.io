Términos de Uso

Última actualización: Noviembre 2025

1. Aceptación de los términos

Al acceder y utilizar Escapify, aceptas cumplir con estos términos de uso. Si no estás de acuerdo con alguna parte de estos términos, no debes utilizar el servicio.

2. Descripción del servicio

Escapify es una plataforma web educativa gratuita que permite a los usuarios crear candados digitales y escape rooms para uso educativo. El servicio se proporciona "tal cual" sin garantías de ningún tipo.

3. Uso permitido

Puedes utilizar Escapify para:

  • Crear candados digitales y escape rooms con fines educativos
  • Compartir tus creaciones con estudiantes y colegas
  • Utilizar las plantillas y recursos proporcionados
  • Exportar e importar tus datos

4. Uso prohibido

No está permitido:

  • Utilizar el servicio para actividades ilegales o no éticas
  • Intentar acceder no autorizado a sistemas o datos
  • Modificar, copiar o distribuir el código sin autorización (a menos que sea código abierto)
  • Utilizar el servicio para enviar spam o contenido malicioso
  • Violar derechos de propiedad intelectual de terceros

5. Propiedad intelectual

El código fuente de Escapify es de código abierto y está disponible bajo licencia. Los candados y escape rooms que crees son de tu propiedad. Al utilizar el servicio, otorgas a Escapify una licencia no exclusiva para mostrar y distribuir tu contenido a través del servicio.

6. Limitación de responsabilidad

Escapify se proporciona "tal cual" sin garantías de ningún tipo, expresas o implícitas. No garantizamos que:

  • El servicio esté siempre disponible o libre de errores
  • Los datos almacenados localmente estén siempre seguros
  • El servicio cumpla con todos tus requisitos específicos

No seremos responsables de ningún daño directo, indirecto, incidental o consecuente derivado del uso o la imposibilidad de usar el servicio.

7. Almacenamiento de datos

Los datos se almacenan localmente en tu navegador. Es tu responsabilidad:

  • Hacer copias de seguridad regulares de tus datos
  • Exportar tus datos antes de limpiar el caché del navegador

No nos hacemos responsables de la pérdida de datos almacenados localmente.

  • Supervisar el uso por parte de estudiantes

Podemos actualizar estos términos de uso ocasionalmente. Los cambios entrarán en vigor al publicarlos en esta página. Te recomendamos revisar estos términos periódicamente.
