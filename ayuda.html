Preguntas Frecuentes

¿Qué es un candado digital?

Un candado digital es una herramienta interactiva que permite crear desafíos o pruebas que requieren un código específico para ser resueltos. Pueden ser numéricos, direccionales, de colores, patrones y más.

¿Cómo puedo crear mi primer candado?

Para crear un candado, ve a la sección "Generador", selecciona el tipo de candado que deseas crear, configura las opciones y guarda. ¡Es así de sencillo!

¿Puedo compartir mis candados?

Sí, puedes compartir tus candados de varias formas: mediante códigos QR, enlaces directos o exportando el historial completo.

¿Los candados son seguros?

Los candados se almacenan localmente en tu navegador y no se comparten con servidores externos. Sin embargo, recuerda que son herramientas para entretenimiento y no para seguridad crítica.

Características de Accesibilidad

En Escapify nos comprometemos a hacer que nuestras herramientas sean accesibles para todos. Hemos implementado numerosas características para garantizar que todos los usuarios puedan disfrutar de nuestra plataforma, independientemente de sus capacidades.

1

ColorADD para Daltónicos

Hemos implementado el sistema ColorADD, un "alfabeto de colores" que permite a las personas con daltonismo identificar correctamente los colores mediante símbolos estandarizados.

¿Qué es ColorADD?

ColorADD® es un lenguaje único, universal, inclusivo y no discriminatorio que permite a las personas con daltonismo identificar los colores, siempre que el color sea un factor de identificación, orientación o elección.

El daltonismo afecta aproximadamente al 8% de los hombres, aproximadamente 350 millones de personas en el mundo. ColorADD utiliza símbolos gráficos estandarizados que representan los colores de forma universalmente reconocida.

Referencia oficial: ColorADD - El Código ColorADD

El Código ColorADD - Símbolos Primarios

Cinco símbolos gráficos representan los Colores Primarios (Azul, Amarillo y Rojo), más el Blanco y Negro:

Azul

Triángulo apuntando hacia la derecha

Amarillo

Barra diagonal de izquierda abajo a derecha arriba

Rojo

Triángulo apuntando hacia la izquierda

Blanco

Cuadrado blanco con borde negro

Negro

Cuadrado negro

Teoría de la Adición de Color

A través del conocimiento adquirido de la "Teoría de Adición de Color", se pueden relacionar los Símbolos del Código e identificar toda la paleta de colores. Los símbolos se combinan para crear colores secundarios y terciarios.

Colores Secundarios

Los colores secundarios se forman combinando los símbolos de los colores primarios:

Verde

Azul + Amarillo

Triángulo derecha + Diagonal

Naranja

Amarillo + Rojo

Diagonal + Triángulo izquierda

Púrpura/Morado

Rojo + Azul

Triángulo izquierda + Triángulo derecha

Blanco, Negro, Gris, Dorado y Plateado

Además de los colores primarios, ColorADD incluye símbolos para blanco, negro, grises y colores metálicos:

Blanco

Cuadrado blanco con borde negro

Negro

Cuadrado negro

Gris Claro

Cuadrado blanco con cuadrado negro interior

Gris Oscuro

Cuadrado negro con cuadrado blanco interior

Dorado

Paréntesis + Amarillo (diagonal)

Plateado

Paréntesis + Gris Claro

Tonos Claros y Oscuros

El blanco y negro aparecen para indicar tonos claros y oscuros. Cuando un símbolo incluye el símbolo de blanco, indica un tono claro. Cuando incluye el símbolo de negro, indica un tono oscuro.

Amarillo Claro

Diagonal + Cuadrado blanco

Azul Oscuro

Triángulo izquierda + Cuadrado negro

Ejemplos de tonos:
  • Tonos claros: Símbolo del color + Símbolo blanco (ej: Amarillo claro = Diagonal + Cuadrado blanco)
  • Tonos oscuros: Símbolo del color + Símbolo negro (ej: Azul oscuro = Triángulo derecha + Cuadrado negro)
  • Grises: Combinaciones de blanco y negro (Gris claro = Blanco con negro interior, Gris oscuro = Negro con blanco interior)
  • Metálicos: Dorado y Plateado usan paréntesis seguidos del símbolo correspondiente
¿Cómo funciona en Escapify?

Cuando activas ColorADD en el panel de accesibilidad, verás símbolos junto a todos los elementos que usan colores para transmitir información:

  • Niveles de dificultad: Los badges de dificultad (Fácil, Medio, Difícil) muestran símbolos ColorADD para identificar el color sin necesidad de verlo
  • Candados de colores: Los botones de colores en candados tipo "Color" muestran sus símbolos ColorADD correspondientes
  • Filtros y botones: Todos los botones que usan color para diferenciación incluyen símbolos ColorADD
Activar ColorADD

Para activar ColorADD, utiliza el panel de accesibilidad pulsando el botón en la parte superior derecha de la pantalla, o usa el atajo de teclado Alt + A.

Una vez activado, los símbolos ColorADD aparecerán automáticamente junto a todos los elementos que usan colores, permitiendo identificar la información sin depender de la percepción del color.

¡El color es para Todos! ColorADD convierte la identificación de colores en "un juego mental", fácil de memorizar y aplicar en situaciones cotidianas.

Más información: Visita el sitio oficial de ColorADD para conocer más sobre este sistema universal: www.coloradd.net

2

Para Discapacidad Visual

Hemos implementado varias características para usuarios con baja visión:

  • Modo de alto contraste para mejorar la legibilidad
  • Opción de texto más grande y espaciado
  • Compatibilidad con lectores de pantalla (ARIA)
  • Navegación por teclado mejorada
  • Enlaces "saltar al contenido" para evitar navegación repetitiva
3

Para Discapacidad Auditiva

Aseguramos que los usuarios con discapacidad auditiva puedan utilizar plenamente nuestra aplicación:

  • Alertas visuales para notificaciones importantes
  • No dependemos del audio para funcionalidades esenciales
  • Opciones para desactivar o personalizar efectos de sonido
4

Para Discapacidad Intelectual

Hemos creado opciones para simplificar la experiencia:

  • Modo de interfaz simplificada que reduce elementos distractores
  • Lenguaje sencillo y claro
  • Pictogramas e iconos descriptivos junto al texto
  • Instrucciones paso a paso para tareas complejas
5

Para Adultos Mayores

Entendemos las necesidades específicas de los usuarios mayores:

  • Modo senior con botones y texto más grandes
  • Mayor espaciado entre elementos interactivos
  • Contraste aumentado para mejor legibilidad
  • Navegación simplificada
6

Para Niños

Hemos creado un modo especial para los más pequeños:

  • Interfaz más colorida y atractiva
  • Animaciones amigables
  • Botones más grandes con efectos visuales
  • Lenguaje adaptado al nivel infantil

Cómo Activar las Funciones de Accesibilidad

Para acceder a estas opciones, simplemente haz clic en el botón ubicado en la parte superior derecha de la pantalla. Esto abrirá el panel de accesibilidad donde podrás activar las funciones que necesites.

También puedes usar el atajo de teclado Alt + A para abrir el panel de accesibilidad.
